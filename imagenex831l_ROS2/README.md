# imagenex831l

ROS2 driver for the [IMAGENEX831L](https://imagenex.com/products/831l-pipe-profiling).

## Getting started

### Prerequisites

Set the IP address of the computer connected to the sensor.

### Running
```
ros2 launch imagenex831l sonar.launch
```
